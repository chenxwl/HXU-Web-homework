# Flex box 布局学习
## 01.认识Flexbox
### 1.Flexbox布局
CSS Flexbox 布局模块
在 Flexbox 布局模块（问世）之前，可用的布局模式有以下四种：

块（Block），用于网页中的部分（节）
行内（Inline），用于文本
表，用于二维表数据
定位，用于元素的明确位置
弹性框布局模块，可以更轻松地设计灵活的响应式布局结构，而无需使用浮动或定位。  
如需开始使用 Flexbox 模型，您需要首先定义 Flex 容器。
